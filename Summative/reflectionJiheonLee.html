REFLECTION

Why is it important for high school students to learn programming?
Programming requires logical and problem-solving mind to build an efficient algorithm according to the need of the writer. Even if some of the students are not interested in computer science, it is important to establish a funcdamental knowledge in programming to grow mathmetical and logical thinking skills in adolescent where a brain is developed the fastest in one's life. Moreover, most of the industry present in the world is evolving with a computational technology, and it is inevitable to operate computers or electronic devices in workplaces after high school students grow up. Therefore, it is important to learn programming not only for the students pursuing that field but also all other students.
What made the work difficult? What were the easy parts?
Part that made the work difficult is the compatibility between CSS and JavaScript. While manipulating styles with JavaScript, it often cause a crash with existing CSS styles of the previous styles manipulated through JavaScript. Thus, I need to analyze both CSS and JavaScript algorithm to solve the problems. Moreover, I designed the webpages as much responsive as possible. In the course of making a responsive design, I should test various combination of CSS properties and values. This work took a lot of time and effort to successfully acheive the responsive webpages. Another thing was the algorithms in clock operation. Since each digit of milliseconds, seconds, and minutes changes a digit at different numbers, I had to design an efficient algorithms to check all the cases and change corresponding elements in HTML.
After completeing the clock and flashcards, the ToDo List came to be easier than the previous tasks. With some transition effects, I could easily create several functions to manipulate the items in the list. Because It doesn't require a complex code for the operation like the clock, It was the easy part of the summative project.
How can I use what I learned in this course in the future?
Although I am not considering to pursue web developing, a website as a medium is highly appreciable. Websites possess exceptionally high accessibility and legibility to appeal myself to public. Therefore, I could build a website for the portfolio of my career. Using JavaScript I learned in this course and incorporating the sense of visual design, I could create a responsive portfolio of my skills developed through post-secondary education and work experiences.
If I were to have more time, what would I improve on my project?
In my opinion, there are not many things to improve functionally, but I would improve visual aspect of my project. If I had more time to work on, I would have meticulously design the pages considering the arrangement of buttons and interfaces. As a result, I could create more visually pleasing website with more completeness. Also, I want to contemplate the code and algorithm I wrote for each task and contrive more efficient and simpler solution for the project.
